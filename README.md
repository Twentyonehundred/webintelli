# WebIntelli

Test WordPress site for WP Engine Intelligent Web Tools.

This repository contains a custom WordPress theme designed to showcase and test WP Engine's intelligent web tools and features.

## Auto-Deployment

This theme automatically deploys to WP Engine via GitHub Actions when changes are pushed to the `main` branch.

## Environment

- **WP Engine Install**: webintell
- **Theme Location**: wp-content/themes/webintelli/
- **Deployment**: Automated via GitHub Actions
